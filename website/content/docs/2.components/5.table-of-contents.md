## Anatomy

```vue
<template>
  <LegoToc>
    <template #title></template>

    <LegoTocLinks v-slot="{ link }" :links="links" />
  </LegoToc>
</template>
```

## API Reference

### LegoTocLinks

Wrapper that provide the information required for the child component.

| Prop  | Default | Types    | Description                   |
| ----- | ------- | -------- | ----------------------------- |
| links | -       | `Link[]` | Links from Nuxt Content's toc |
